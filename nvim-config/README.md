A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

## Installation

1. Clone this repository to your nvim config directory:
   ```bash
   git clone <your-repo-url> ~/.config/nvim
   ```

2. Start Neovim and let LazyVim install all plugins:
   ```bash
   nvim
   ```

## Features

- Multiple colorschemes available (everforest, gruvbox, catppuccin, etc.)
- Transparent background support
- Theme hot-reloading capability
- Neo-tree file explorer
- Disabled news alerts
- Custom keymaps and options
